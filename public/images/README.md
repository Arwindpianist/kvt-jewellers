# Image Directory

Place your product images here:

- `gold-coins.jpg` - For gold coins category
- `gold-bars.jpg` - For gold bars category  
- `jewelry.jpg` - For jewelry category
- `store.jpg` - For store/showroom images

After adding images, update `lib/image-placeholders.ts` to use local paths:
- Change from Unsplash URLs to `/images/your-image.jpg`
